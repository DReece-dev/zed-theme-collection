## Installation

To install and use these themes in Zed, follow these simple steps.

### Manual Installation

1.  **Locate your Zed themes directory.**
    - **macOS & Linux:** `~/.config/zed/themes/`
    - **Windows:** `%APPDATA%/zed/themes/`

2.  **Copy the theme files.**
    Copy the `.json` theme files from this repository into the Zed `themes` directory you located in the previous step.

3.  **Select the theme in Zed.**
    - Open Zed.
    - Open the theme selector by pressing `Cmd + K` then `Cmd + T` (on macOS) or `Ctrl + K` then `Ctrl + T` (on Linux/Windows).
    - You should now see the new themes in the list. Select the one you want to use.

## Contributing

Contributions are welcome! If you have a theme you'd like to add to this collection, please follow these steps:

1.  **Fork the repository.**
2.  **Create a new branch:** `git checkout -b your-theme-name`
3.  **Add your theme's `.json` file** to the root directory.
4.  **Update this README.md** to include your theme in the [Themes](#themes) section with a screenshot and a brief description.
5.  **Submit a pull request.**

When creating new themes, you can refer to the official Zed documentation on [theme creation](https://zed.dev/docs/extensions/themes).
